# RHEL servers
If Kernel version: **4.18.0-553.34.1.el8_10.x86_64**
### Format: major.minor.patch_version.el<Enterprise Linux>RHEL_Version.Architecture
from above example    
     **4**  -> major version  
     **18**  -> Minor Version  
     **0**  -> patch version  
     **553.34.1** -> build version  
     **el8_10** -> RHEL 8.10  
     **886_64** -> Architecture
